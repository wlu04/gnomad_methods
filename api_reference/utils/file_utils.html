async gnomad.utils.file_utils.parallel_file_exists_async(fpaths, parallelism=750)[source]¶

Check whether a large number of files exist.

Created for use with hail Batch jobs. Normal file_exists function is very slow when checking a large number of files.

Parameters

  • fpaths (List[str]) – List of file paths to check. Files can be in local or Google cloud storage.

  • parallelism (int) – Integer that sets parallelism of file existence checking task. Default is 750.

Return type

Dict[str, bool]

Returns

Dictionary of file paths (str) and whether the file exists (boolean).
